# Telegram‑бот для скачивания видео из TikTok по ссылке.

Бот принимает ссылку на TikTok и отправляет видеофайл прямо в Telegram без водяного знака (если источник это поддерживает).

---

## Технологии

- **Python 3.10+**
- **aiogram 3** — асинхронный фреймворк для Telegram ботов
- **aiohttp** — асинхронные HTTP запросы
- **asyncio** — асинхронная обработка задач
- **python-dotenv** — загрузка переменных окружения из `.env`

---

## Возможности

- Скачивание видео из TikTok по ссылке
- Асинхронная обработка запросов
- Использование нескольких API провайдеров
- Работа в личных сообщениях и группах
- Отправка видео прямо в Telegram
- Автоматический fallback если один сервис недоступен

---

## Как работает бот

1. Пользователь отправляет ссылку на TikTok
2. Бот обращается к API сервисам:
   - `tikwm`
   - `lovetik`
3. Получает прямую ссылку на видео
4. Отправляет видео в Telegram чат
